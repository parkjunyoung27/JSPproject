# JSPproject
JSP를 이용해서 게시판사이트를 만들었습니다.

# Skills

1. 언어 - java, HTML5, CSS, javascript, jquery, ajax   
2. 툴 - 이클립스, dbeaver 
3. 서버 - tomcat
4. 호스팅 - 라즈베이파이를 이요해 라즈비안 OS에 배포<br> 

# 게시판 기능

1. 글쓰기 
2. 사진 올리기 
3. 로그인
4. 회원가입
5. ID, PW찾기
6. 검색
7. 상세보기 - 좋아요기능
8. 댓글
